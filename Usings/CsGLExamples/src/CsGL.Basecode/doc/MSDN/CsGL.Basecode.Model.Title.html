Model.Title Property

Model-specific title. 

[Visual Basic]
Overridable Public ReadOnly Property Title As String
[C#]
public string Title {virtual get;}

Remarks

Override this to provide a title for your application for use in the basecode's help screen. If you're not using the basecode's help screen, you can either disregard this property, or use it how you like.

Example

In your inherited Model: 

            public override string Title {
            	get {
            		return "My OpenGL Lesson";
            	}
            }
