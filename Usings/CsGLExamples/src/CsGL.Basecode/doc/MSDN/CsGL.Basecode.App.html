Remarks

The App class has methods to start and stop CsGL applications and to provide useful CsGL application functionality.

Call Run to start a CsGL application using the appropriate inherited Model. Call Terminate to forcibly end a running CsGL application. Calling ToggleFullscreen toggles between your supplied Windows Form and the fullscreen form. Call ResetFramerate to reset the application's current framerate calculations.

This class has numerous properties to get or set information about the current state of the application.

You cannot create an instance of this class.
