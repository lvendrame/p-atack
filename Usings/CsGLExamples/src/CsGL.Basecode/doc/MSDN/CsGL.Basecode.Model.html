CsGL.Basecode

Model Class

Provides a template for a CsGL application using the CsGL Basecode. It provides methods for setting up the application, initializing, drawing, and user input. Many of the provided methods can and should be overridden in your inherited model.

Remarks

The Model class is the template for all applications using the CsGL Basecode. It provides basic functionality on its own, to allow for rapid development, however, by overriding the appropriate methods you can modify and extend your application's functionality over what is provided by default.
