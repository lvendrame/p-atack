Model.WindowTitle Property

Model-specific window title. 

[Visual Basic]
Overridable Public ReadOnly Property WindowTitle As String
[C#]
public string WindowTitle {virtual get;}

Remarks

Override this to provide a title for your application in its title bar. This title is not used in the basecode's help screen, only in the application's title bar. If this is not supplied in your model, the basecode will use the Title property for the title bar.

Example

In your inherited Model: 

            public override string WindowTitle {
            	get {
            		return "My OpenGL Lesson's Title Bar!";
            	}
            }
