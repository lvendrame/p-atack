CsGL.Basecode

CsGL.Basecode Namespace

Namespace hierarchy


The CsGL.Basecode Namespace Provides A Simple, Yet Feature-Rich Framework For Developing OpenGL Applications Using CsGL (The .Net Wrapper For OpenGL).

Classes

ClassDescription
App Provides static (Shared in Visual Basic) methods and properties to manage a CsGL application. It provides methods and properties to start and stop a looped CsGL application and to manage its runtime state. This class cannot be inherited.
AppForm Provides a default Windows Form for basecode applications.
HelpForm Provides a Windows Form for example information and help.
HelpFormDataGrid Custom DataGrid to enable ScrollBar and override editing mode. This is FUGLY!
HighResolutionTimer Encapsulates the functionality of the platform's highest resolution timer available.
Model Provides a template for a CsGL application using the CsGL Basecode. It provides methods for setting up the application, initializing, drawing, and user input. Many of the provided methods can and should be overridden in your inherited model.
SetupForm Provides a default form for end-user application setup.
View Provides an OpenGL view control for your application's form.
